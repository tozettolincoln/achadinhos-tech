# Landing Page - Achados Tech Tz

Landing page estática para converter visitantes para o Grupo de Promoções Achados Tech Tz no WhatsApp.

## Link oficial do grupo

`https://chat.whatsapp.com/GmVTplmLx88FddVR1oLpQT`

O link está aplicado nos CTAs:

- `Entrar no grupo grátis do WhatsApp` no topo
- `Entrar no grupo grátis do WhatsApp` no CTA final

Os links abrem na mesma aba para funcionar melhor em navegadores internos, incluindo o Instagram.

## Identidade visual

- Nome público: `Achados Tech Tz`.
- Paleta: preto tech, roxo noite, roxo elétrico, lilás neon, verde WhatsApp, amarelo achado, branco suave e cinza técnico.
- Direção: marca tech confiável, com curadoria, testes, comparativos e alertas.
- Verde reservado para CTAs ligados ao WhatsApp.
- Amarelo usado em selos de revisão e transparência.

## Assets usados

Assets principais da experiência:

- `avatar-linc-tech-tz-v2.png`: persona usada no perfil e nos metadados sociais.
- `whatsapp-icon-attached.png`: ícone fornecido para os CTAs do WhatsApp.

## Arquivos principais

- `index.html`: landing page principal.
- `styles.css`: tema visual Achados Tech Tz.
- `privacy.html`: política de privacidade e consentimento para WhatsApp.
- `SECURITY.md`: checklist de segurança.
- `_headers`: cabeçalhos para hosts que suportam esse formato.
- `.nojekyll`: compatibilidade com GitHub Pages.

## URL pública esperada

`https://tozettolincoln.github.io/achadinhos-tech/`

Os metadados sociais (`canonical`, `og:url`, `og:image` e `twitter:image`) apontam para essa URL.

## Antes de publicar

- Conferir a publicação no GitHub Pages após o deploy.
- Confirmar que os dois CTAs navegam para o convite oficial na mesma aba.
- Manter a promessa conservadora: curadoria e filtro, sem menor preço garantido, estoque garantido ou urgência falsa.
