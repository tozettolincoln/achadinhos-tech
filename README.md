# Landing Page - Achadinhos Tech [01]

Página de entrada focada em uma única conversão: levar o visitante para o grupo oficial do WhatsApp.

## Link Oficial Do Grupo

`https://chat.whatsapp.com/GmVTplmLx88FddVR1oLpQT`

O link está aplicado no botão principal:

`Entrar no grupo grátis`

## Arquivos Principais

- `index.html`: página de entrada.
- `styles.css`: tema visual preto/verde focado em WhatsApp.
- `privacy.html`: política de privacidade.
- `SECURITY.md`: checklist de segurança.
- `_headers`: cabeçalhos para hosts que suportam esse formato.
- `.nojekyll`: compatibilidade com GitHub Pages.

## URL Pública Esperada

`https://tozettolincoln.github.io/achadinhos-tech/`

Os metadados sociais (`canonical`, `og:url`, `og:image` e `twitter:image`) já apontam para essa URL.

## Antes De Publicar

- Trocar `contato@example.com` por um e-mail real em `privacy.html` e `.well-known/security.txt`.
- Conferir a publicação no GitHub Pages após o deploy.
